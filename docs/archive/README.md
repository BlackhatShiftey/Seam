# SEAM Documentation Archive

This folder separates inactive documentation from the active SEAM docs.

Use this folder for:

- old handoff files
- superseded setup notes
- outdated implementation notes
- historical coding artifacts that are useful for traceability but should not guide new work

Do not use this folder for current operator instructions. If an archived note becomes relevant again, rewrite the useful parts into an active doc and point back to the related `HISTORY.md` entry instead of restoring stale prose wholesale.
